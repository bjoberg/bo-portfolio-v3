# BrettOberg.com

Portfolio website.

## Installation

1. Install node.js (v5.x.x)
2. Install npm (v3.x.x)
3. Clone this repository
4. Within your command line, navigate to your local project directory and run **`npm install`**
5. Run **`node src/app.js`**
